# IMDB Scrapper
Scrap IMDB movies using [BeautifulSoup](https://www.crummy.com/software/BeautifulSoup/bs4/doc/).

# Execute
`python imdb-scrapper.py`

# Dependencies
* Python
* Beautiful Soup
* Requests
